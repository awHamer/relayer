# Relayer Documentation

Built with [Astro Starlight](https://starlight.astro.build/).

## Development

```bash
pnpm install
pnpm dev       # start dev server at localhost:4321
```

## Build

```bash
pnpm build     # output to dist/
pnpm preview   # preview production build
```

## Deployment

The site is static HTML. Deploy to any static host (Vercel, Cloudflare Pages, Netlify).

## Structure

Documentation pages live in `src/content/docs/`. Sidebar is configured in `astro.config.mjs`.
